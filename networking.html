Networking Concepts

DNS:
For my domain to be resolved to an IP, first someone enters the URL (https://cvatter.github.io/Network-Project-1/about.html).
Next the browser will check router, DNS, and OS cache to figure out if the IP has already been visited/is already known.
If not known, then the request goes through a recursive resolver that will query through IP addresses until it finds my website.
Next the resolver gets the IP address for my website.
Finally my browser connects to the IP.
